# 沈家本研究院官网 — 完整版开发需求文档

## 0. 项目总述
- **项目名称**：沈家本研究院官方网站（全自动法律数据学术中台版）
- **项目定位**：庄重学术政务官网 + 全自动无人值守法律数据聚合平台
- **核心特性**：全站动态数据 100% API驱动、零人工录入、零人工审核、全自动更新
- **开发模式**：前后端完全分离
- **终端要求**：PC端 + 移动端 100%自适应响应式

## 1. 开发分工

### 1.1 前端全域
1. 全站所有页面UI、布局、样式、自适应适配
2. 全站所有前端交互监听逻辑
3. 所有 ECharts 数据可视化图表开发与渲染
4. 完整后台管理系统前端页面
5. 表单提交、留言、报名、导出功能前端实现
6. 全站统一加载、空数据、报错兜底页面样式
7. 最终交付：完整前端源码 + 接口对接文档 + 部署说明

### 1.2 后端全域
1. 服务器搭建、数据库搭建、后端程序开发、长期运维
2. 五大板块7×24H全自动数据引擎
3. 全套标准化API接口
4. 数据源适配、反爬、改版适配、异常重试、日志监控
5. 数据合规过滤、过期清理、脏数据剔除、数据安全

### 1.3 全包托管
域名选购注册、解析配置、服务器选型、部署、HTTPS、ICP备案协助、全站上线调试

## 2. 全站UI统一规范

### 2.1 整体风格
简约、庄重、学术、政务、法治厚重风；禁止花哨渐变、网红风、卡通、高饱和亮色

### 2.2 全站色值
| 用途 | 色值 | 说明 |
|------|------|------|
| 主色（法治深蓝） | #203B64 | 导航、主按钮、标题强调、图表主色、页脚底色 |
| 辅助浅蓝 | #6B84A8 | 次级按钮、边框、次要标签 |
| 文史点缀古铜色 | #C89959 | 仅Banner、小标题点缀、荣誉标签 |
| 正文 | #222222 | 正文文本 |
| 次级 | #666666 | 次级文本 |
| 辅助 | #999999 | 辅助文本 |
| 页面底色 | #F8F9FA | 页面背景 |
| 卡片白 | #FFFFFF | 卡片背景 |
| 分割线 | #E5E7EB | 分割线 |
| 新增/正常 | #279E68 | 状态绿 |
| 处罚/警示 | #D83030 | 状态红 |
| 进行中 | #E69523 | 状态橙 |
| 过期/失效 | #AAAAAA | 状态灰 |

### 2.3 字体规范
- 标题：思源宋体 / 系统宋体（厚重文史感）
- 正文、表格、按钮、数据：思源黑体 / 微软雅黑
- 字号：大标题28-32px、板块标题20-24px、卡片标题16-18px、正文14-15px、备注12-13px

### 2.4 布局规范
- PC端：版心最大1200px居中
- 移动端：左右16px内边距
- 卡片：圆角6px、轻微浅阴影、内边距20px

### 2.5 动效规范
- 允许：导航吸顶、卡片hover微上浮、淡入淡出、loading
- 禁止：粒子动画、视差滚动、闪烁、弹跳、大面积动态特效

### 2.6 全局兜底
- 加载中：「数据加载中…」
- 无数据：「暂无匹配数据」
- 异常：「数据加载异常，请稍后重试」

## 3. 全站页面模块清单

### 阶段一：静态基础页面
1. 首页（预留五大动态数据接口区块）

6. 商事调解中心专题页
7. 活动专栏+活动预约表单
8. 联系我们（地址、电话、留言表单、地图）
9. 公共头部导航、移动端汉堡菜单、统一合规页脚

### 阶段二：五大动态数据核心页面
