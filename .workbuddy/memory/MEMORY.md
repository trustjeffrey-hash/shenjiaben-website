## 技术栈
- 前端：纯 HTML/CSS/JS + ECharts，无框架
- 后端：Node.js 零依赖（内置 http/fs/path/crypto/url）
- 数据库：JSON 文件数据库（自写 db.js 模块）
- 部署：Render.com 免费托管

## 页面结构
- src/frontend/ — 前台页面（15 个 HTML 页面 + 公共 CSS/JS）
- src/admin/ — 后台管理系统
- src/backend-js/ — Node.js 后端（server.js + db.js + seed.js）
- dist/ — CloudStudio 部署输出（与 src/frontend 同步）
- docs/ — 文档

## 关键约定
- 所有动态内容 100% API 驱动，禁止硬编码假数据
- 全站加载/空/异常兜底统一
- 交互监听：防抖搜索、筛选联动、分页无刷新、图表联动
- 前后端字段名必须严格对齐（已修复 historical 不匹配问题）
