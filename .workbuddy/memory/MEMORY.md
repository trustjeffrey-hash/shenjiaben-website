# 沈家本研究院官网 — 项目记忆

## 项目概况
- 项目名称：沈家本研究院官方网站
- 定位：庄重学术政务官网 + 全自动法律数据聚合平台
- 开发模式：前后端完全分离
- 终端：PC + 移动端 100% 自适应

## UI 规范（严格遵循）
- 主色：法治深蓝 #203B64
- 辅色：浅蓝 #6B84A8
- 点缀：古铜色 #C89959（仅 Banner/标签）
- 标题字体：思源宋体
- 正文字体：思源黑体/微软雅黑
- 卡片：圆角 6px、浅阴影、内边距 20px
- 版心：PC 1200px 居中，移动端 16px 内边距

## 技术栈
- 前端：纯 HTML/CSS/JS + ECharts，无框架
- 后端：待开发（Python/Node.js）
- 部署：待选型

## 页面结构
- src/frontend/ — 前台页面（9 个页面 + 公共 CSS/JS）
- src/admin/ — 后台管理系统
- src/backend/ — 后端（待开发）
- docs/ — 文档

## 关键约定
- 所有动态内容 100% API 驱动，禁止硬编码假数据
- 全站加载/空/异常兜底统一
- 交互监听：防抖搜索、筛选联动、分页无刷新、图表联动
