# 2026-06-27 工作日志

## 凌晨 — 修复种子数据未来日期

- 用户截图反馈：数据平台显示"2026年12月"等未来日期（当前为2026年6月）
- 根因：seed.js 中 `Math.floor(Math.random() * 12)` 在 year=2026 时可能生成 7-12 月
- 修复方案：所有种子数据日期生成增加条件判断——当年份为 2026 时，月份上限锁定 Math.floor(Math.random() * 6)（1-6月）
- 影响4个数据表：lawyer_discipline / legislation_projects / judicial_cases / judicial_policies
- 采集器 simulate() 此前已正确限制为 Math.floor(Math.random()*6)，无需修改
- commit: c48ba12，已推 GitHub
